# Leasing software 

### Prerequisites
```
pip install -r requirements.txt 
```

### Installing
```
cd v0.2
cd result
cd backend
python app_final.py
cd ../ 
cd frontend
npm start
```

## Built With
* [Flask](http://flask.pocoo.org/) 
* [NodeJS](https://nodejs.org/) 
* [ReactJS](https://reactjs.org/)
